INSTRUCCIONES PARA IMPORTAR LA BASE DE DATOS (MARIADB)
======================================================

Para poder correr el proyecto y ver todo lo que tenemos cargado en la base de datos, sigue estos pasos:

1. CONFIGURACIÓN EN TU SERVIDOR LOCAL (MARIADB):
   - Crea una base de datos vacía en tu servidor MariaDB local.

2. CONFIGURACIÓN DEL ARCHIVO .ENV:
   - Configura las credenciales de conexión en tu archivo `.env` para que apunten a tu base de datos local:
     
     DB_CONNECTION=mariadb
     DB_HOST=127.0.0.1
     DB_PORT=3306
     DB_DATABASE=taller_1
     DB_USERNAME=tu_usuario_de_mariadb
     DB_PASSWORD=tu_contraseña_de_mariadb

3. INICIALIZACIÓN DEL PROYECTO:
   - Ejecuta los siguientes comandos en la terminal:
     
     composer install
     npm install
     npm run dev
     php artisan serve
